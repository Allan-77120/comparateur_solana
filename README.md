 Solana APY Comparator
Un comparateur d'APY pour les protocoles DeFi sur Solana
Solana APY Comparator est une application qui permet aux utilisateurs de comparer les rendements (APY) des différents protocoles DeFi (staking, lending, farming, etc.) sur la blockchain Solana. L'objectif est d'aider les investisseurs à prendre des décisions éclairées en visualisant les meilleures opportunités de rendement.

Fonctionnalités

✅ Comparaison en temps réel des APY des protocoles DeFi sur Solana
✅ Filtrage par catégorie (Staking, Lending)
✅ Classement des protocoles par APY, TVL (Total Value Locked) ou risque
✅ Interface intuitive et responsive (Web & Mobile)

Technologies utilisées

Frontend : React.js / Next.js (TypeScript)
Backend : Node.js / Express (ou API externe comme DEfiLLama)
Blockchain : Solana 
Base de données : MongoDB
UI/UX : Tailwind CSS
API :  DeFi Llama
